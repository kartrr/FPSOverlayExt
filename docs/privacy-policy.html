Privacy Policy for FPS & Performance Overlay

Last updated: September 01, 2025

Thank you for using FPS & Performance Overlay. This Privacy Policy is to inform you about how we handle data in our browser extension.

Data Collection and Usage

The Extension does not collect, store, or transmit any personal information, browsing history, or data from the web pages you visit.

The Extension only accesses two categories of data, all of which remain locally within your browser:

  • Browser Performance Data: To perform its core function, the Extension uses built-in browser APIs to measure performance. These include:
    • Frame Timestamps (performance.now): to calculate FPS (Frames Per Second) and frame time.
    • JS Heap Memory Usage (performance.memory): to display information about JavaScript memory usage.
    • Long Tasks (PerformanceObserver): to track tasks that may slow down the page.
    This data is anonymous, technical, and is used solely for calculations and display in the overlay.
  • Page Structure (DOM): To count the total number of DOM elements on a page, the Extension analyzes the structure of the web page.

    Important: The Extension only counts the number of elements. It does not read, save, or transmit the content of these elements, such as text, passwords, form data, or any other personal information.

Data Storage

The Extension only stores your settings. For example, which metrics to display (FPS, memory, etc.) and the threshold values you set.

  • These settings are saved locally on your computer using localStorage or are synced across your devices via your Google Account using chrome.storage.sync.
  • As the developers, we have no access to this data. The storage of settings is solely for your convenience, so you don't have to reconfigure the overlay every time.

Permissions

In the manifest.json file, the Extension requests access to <all_urls>. This permission is necessary for the following reason:

  • To allow you to activate the overlay on any page you wish to test or debug using the hotkey (Ctrl+Alt+F).
  • This permission is not used to track your activity, collect browsing history, or read the content of web pages.

Data Sharing and Third-Party Services

We do not transmit any of the above-mentioned data over the network. Your browsing history, website content, and error information remain on your device.

Changes to This Privacy Policy

We may update our Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page.
